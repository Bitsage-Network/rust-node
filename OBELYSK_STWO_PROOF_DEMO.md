## Technical Details

### M31 Field Arithmetic
- **Prime**: p = 2³¹ - 1 = 2147483647
- **Reduction**: Uses Mersenne property for fast modular arithmetic
- **Range Check**: All field elements must be < 2147483647

### Circle STARK Protocol
1. **Trace Commitment**: Merkle tree root of execution trace
2. **FRI Layers**: Fast Reed-Solomon IOP for proximity testing
3. **OODS**: Out-of-Domain Sampling for constraint verification
4. **PoW**: Proof-of-Work grinding for security amplification

### Security Calculation
```
security_bits = log_blowup_factor × n_queries + pow_bits
             = 3 × 16 + 16 + additional_constraints
             ≈ 128 bits
```
